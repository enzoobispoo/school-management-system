# Sistema de Gestão Escolar

Sistema completo de gestão escolar desenvolvido com foco em performance, usabilidade e escalabilidade. Ideal para escolas, cursos e instituições de ensino que desejam centralizar suas operações em uma única plataforma moderna.

---

## Funcionalidades

### Alunos
- Cadastro completo (nome, email, telefone, CPF, endereço)
- Matrícula em múltiplos cursos
- Histórico financeiro por aluno
- Status de pagamento (pago, pendente, atrasado)

### Cursos & Turmas
- Criação de cursos com categoria, duração e valor mensal
- Gerenciamento de turmas (horários, professor, capacidade)
- Controle de alunos por turma

### Professores
- Cadastro de professores
- Associação com cursos
- Agenda completa por turma
- Controle de status (ativo/inativo)

### Financeiro
- Controle de mensalidades
- Status: pago, pendente, atrasado
- Registro de pagamentos (Pix, dinheiro, cartão, etc.)
- Geração automática de mensalidades
- Exclusão e gestão de cobranças

### Relatórios
- Receita ao longo do tempo
- Crescimento de alunos
- Cursos mais populares
- Insights de negócio (retenção, ticket médio, metas)

### Assistente com IA
- Consulta inteligente sobre alunos, pagamentos e relatórios
- Interface de chat integrada ao dashboard

---

## Tecnologias

- **Frontend:** Next.js (App Router) + TypeScript  
- **UI:** TailwindCSS + shadcn/ui  
- **Backend:** API Routes (Next.js)  
- **Banco de Dados:** PostgreSQL (Neon)  
- **ORM:** Prisma  
- **Gráficos:** Recharts
